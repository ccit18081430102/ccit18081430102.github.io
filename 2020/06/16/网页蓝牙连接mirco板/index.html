网页蓝牙连接mirco:bit

本章任务

  1. 了解网页蓝牙和mirco:bit
  2. 测试板子连接
  3. 通过网页蓝牙连接
  • 网页蓝牙

网页蓝牙：通过渐进式 Web 应用（Progressive Web Apps）技术，你可以开发成熟的 Web 应用。 得益于大量新规范和新功能，以前需要在本机执行的应用，现在可以基于 Web 实现。

  • Micro:bit

Micro:bit是由英国BBC公司推出的面向青少年编程教育的微型计算机，这一块小板子上集成了加速度传感器，磁力传感器，两个可编程按钮，25个单色led，蓝牙等常用传感器设备，采用micro usb口供电，可外接电池盒，在底部还有多个环孔连接器，可用于控制外接设备，很适合小孩子学习编程，非常容易入门。

接下来我们就来测试板子是否和本机连接正常。

  1. 首先，下载测试程序网站。

    https://makecode.microbit.org/v0/89975-20684-84568-20127

  2. 将下载好的程序拖进micro:bit自带的盘里。

  3. 观察现象。

    板子上如果滚动显示“Hi！”就说明和本机以连接成功,当蓝牙连接成功后会显示C，断开连接会显示D。
